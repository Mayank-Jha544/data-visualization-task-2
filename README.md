# Data Visualization Task 2

## Objective
To create charts and understand sales data.

## Dataset
Superstore sales dataset.

## Tool Used
- Power BI

## Charts Created
- Sales by Category
- Sales by Region
- Sales over Time
- Profit by Category

## Key Insights
- Technology has highest sales
- West region performs best
- Sales are increasing over time
- Some categories have lower profit

## Files
- Power BI file (.pbix)
- PDF report
- Dataset (Superstore.csv)

## Conclusion
Data visualization helps to understand data and find useful insights.
